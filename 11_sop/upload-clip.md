# SOP: Upload Clip

## เป้าหมาย

อัปโหลดคลิปอย่างเป็นระบบ และเก็บข้อมูลเพื่อวัดผล

## ก่อนอัปโหลด

- [ ] ตรวจชื่อไฟล์
- [ ] ตรวจเสียง
- [ ] ตรวจ caption
- [ ] ตรวจ hashtag
- [ ] ตรวจว่าคลิปไม่ยาวเกินจำเป็น

## Caption Structure

```text
[Hook สั้น]

[Context 1 บรรทัด]

[Call to action ถ้ามี]

#TTLIVE #[ชื่อเกม] #TikTokLive
```

## หลังอัปโหลด

1. เก็บ link คลิป
2. จดใน `09_reports/clip-performance/`
3. กลับมาดูผลที่ 24 ชั่วโมง
4. กลับมาดูผลที่ 7 วัน

## สิ่งที่ต้องเก็บ

- วันที่โพสต์
- platform
- link
- hook ที่ใช้
- views/likes/comments/shares
- lesson
