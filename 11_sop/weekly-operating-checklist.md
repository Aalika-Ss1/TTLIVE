# Weekly Operating Checklist

ใช้เป็น checklist ประจำสัปดาห์สำหรับ TTLIVE เมื่อทำคนเดียว

## Weekly Planning

- [ ] เลือกเกมหลักของสัปดาห์
- [ ] เลือกวันไลฟ์
- [ ] ตรวจว่าแต่ละ live มีเวลาอย่างน้อย 1 ชั่วโมง
- [ ] สร้างตารางรายสัปดาห์
- [ ] เตรียม caption ประกาศ
- [ ] เตรียมภาพตารางหรือภาพก่อนขึ้นไลฟ์ถ้าทำทัน

## Before Each Live

- [ ] เปิด checklist ก่อน live
- [ ] ตั้งชื่อ live
- [ ] เตรียม overlay หรือ starting soon
- [ ] ตรวจไมค์/เสียงเกม/เน็ต
- [ ] เตรียม note สำหรับจด clip candidate

## During Live

- [ ] ไลฟ์อย่างน้อย 1 ชั่วโมง
- [ ] จดช่วงที่คนดูตอบสนองดี
- [ ] จดช่วงที่มีจังหวะตลก/ลุ้น/สอน/พลาด
- [ ] จดปัญหาที่เกิดขึ้น

## After Live

- [ ] กรอก live report
- [ ] เพิ่ม clip candidate ลง log
- [ ] เลือก candidate ที่ดีที่สุด 1 อัน
- [ ] วางแผนตัดคลิป

## Clip Work

- [ ] ตัดคลิป 20-60 วินาที
- [ ] เขียน caption
- [ ] อัปโหลด
- [ ] บันทึก link
- [ ] กลับมาดูผล 24 ชั่วโมง

## Weekly Review

- [ ] ดูว่า live ไหนดีที่สุด
- [ ] ดูว่าเวลาไหนดีที่สุด
- [ ] ดูว่า clip ไหนดีที่สุด
- [ ] เลือกสิ่งที่จะปรับสัปดาห์หน้า

## ถ้าเวลาน้อยมาก

ทำแค่นี้ก่อน:

- [ ] ไลฟ์ 1 ครั้งอย่างน้อย 1 ชั่วโมง
- [ ] กรอก live report สั้น
- [ ] จด clip candidate 1 ช่วง

## ถ้ามีเวลาเพิ่ม

- [ ] ตัดคลิป 1-3 คลิป
- [ ] ทำภาพ starting soon
- [ ] ทำ recap หลังไลฟ์
- [ ] ทดลองเวลาไลฟ์ใหม่ 1 อย่าง
