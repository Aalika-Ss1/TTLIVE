# SOP: Create Weekly Schedule

## เป้าหมาย

สร้างตารางไลฟ์ประจำสัปดาห์ให้พร้อมโพสต์

## ขั้นตอน

1. เปิด `DASHBOARD.md`
2. ตรวจเกมและข้อมูลจาก `07_data/games.json`
3. ใช้ template จาก `05_templates/weekly-schedule/schedule-template.md`
4. สร้างไฟล์ใหม่ใน `03_live_operations/schedules/`
5. ถ้าต้องทำเป็นรูปภาพ ให้ใช้ layout จาก `05_templates/weekly-schedule/schedule-layout.json`
6. เก็บไฟล์ final ใน `08_outputs/schedules/`
7. หลังจบสัปดาห์ สรุปผลใน `09_reports/weekly/`
