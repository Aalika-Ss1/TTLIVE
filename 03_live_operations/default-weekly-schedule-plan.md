# Default Weekly Schedule Plan

ตารางนี้เป็นแผนตั้งต้น ไม่ใช่ข้อบังคับตายตัว ใช้เป็น baseline เพื่อทดลอง 2-4 สัปดาห์แล้วค่อยปรับจากผลจริง

## หลักคิด

- Soul Land: Awakening World เป็นเกมหลักช่วงนี้ เพราะกำลังเล่นทุกวันและเกมเพิ่งเปิด
- Golden Spatula ควรอยู่ช่วงเช้า 3 วัน เพื่อรักษาฐานคนดูเดิมที่คุ้นเวลา 07:00-09:00
- รอบเย็นควรมีประมาณ 4 วัน เพื่อไม่ให้หนักเกินไปเมื่อยังทำงานประจำ
- ศุกร์ เสาร์ อาทิตย์ ต้องกันเวลาบางส่วนไว้พัก ตัดคลิป และสรุปผล
- Naraka: Bladepoint เหมาะกับกลางคืนหรือเสาร์-อาทิตย์ เพราะต้องใช้เวลายาวและเป็นช่วงเรียนรู้เกม

## Recommended Baseline

| วัน | เช้า 07:00-09:00 | เย็น 19:00 เป็นต้นไป | หมายเหตุ |
| :---: | :--- | :--- | :--- |
| จันทร์ | พัก / วางแผน | พัก / เตรียมตาราง | ใช้เป็นวันตั้งระบบและดูงานประจำ |
| อังคาร | Golden Spatula | Soul Land: Awakening World | เริ่มสัปดาห์ด้วยเกมหลักช่วงนี้ |
| พุธ | Golden Spatula | LoL Wild Rift หรือ Soul Land | สลับเกมเพื่อไม่ให้ช่องจำเจ |
| พฤหัสบดี | Golden Spatula | พัก / ตัดคลิป / optional Soul Land | กันแรงไว้ปลายสัปดาห์ |
| ศุกร์ | พัก | Soul Land หรือ LoL Wild Rift แบบเบา | หลังไลฟ์เลือก clip candidate |
| เสาร์ | พัก / ตัดคลิป | Naraka: Bladepoint รอบยาว | เหมาะกับไลฟ์ 2-4 ชั่วโมง |
| อาทิตย์ | Weekly review / ตัดคลิป | Optional Soul Land / พัก | ใช้สรุปผลและเตรียมสัปดาห์ถัดไป |

## Evening Slot Options

ถ้าต้องการรอบเย็น 4 วัน ให้ใช้ชุดนี้ก่อน:

- อังคาร: Soul Land
- พุธ: LoL Wild Rift หรือ Soul Land
- ศุกร์: Soul Land หรือ LoL Wild Rift
- เสาร์: Naraka: Bladepoint

ถ้าสัปดาห์ไหนมีแรงเพิ่ม:

- เพิ่มพฤหัสบดีเย็นเป็น Soul Land แบบสั้น 1 ชั่วโมง
- หรือเพิ่มอาทิตย์เย็นเป็น recap / community live

## Minimum Version

ถ้าสัปดาห์ไหนงานประจำยุ่ง ให้เหลือแค่นี้:

- Golden Spatula เช้า 2 รอบ
- Soul Land เย็น 2 รอบ
- Naraka หรือ Wild Rift 1 รอบในเสาร์/อาทิตย์
- ตัดคลิปอย่างน้อย 1 คลิป

## Review Rule

ทดลองตารางนี้ 2-4 สัปดาห์ แล้วดู:

- เวลาไหนมีคนดูมากกว่า
- เกมไหนทำให้คนคุยเยอะกว่า
- ไลฟ์ไหนตัดคลิปได้ง่ายกว่า
- วันไหนทำแล้วเหนื่อยน้อยที่สุด
