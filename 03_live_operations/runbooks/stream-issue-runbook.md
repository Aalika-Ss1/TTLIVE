# Stream Issue Runbook

ใช้เมื่อเกิดปัญหาระหว่างไลฟ์

## เน็ตหลุด

1. แจ้งคนดูถ้ายังพิมพ์ได้
2. ลด bitrate หรือ quality
3. รีสตาร์ทโปรแกรมไลฟ์ถ้าจำเป็น
4. บันทึกเหตุการณ์ใน report หลังไลฟ์

## เสียงไม่มี

1. ตรวจ input mic
2. ตรวจ mute ในโปรแกรมไลฟ์
3. ตรวจเสียงเกมและเสียงระบบ
4. ขอให้ moderator ช่วยยืนยันใน chat
