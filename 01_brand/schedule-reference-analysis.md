# Schedule Reference Analysis

อ้างอิงจากภาพตารางไลฟ์ Golden Spatula เดิม

## สิ่งที่ควรรักษาไว้

- ตารางกลางแบบแผ่นป้ายหรือกระดาษ ทำให้อ่านง่าย
- แบ่งคอลัมน์ชัดเจน: วัน, เกม, เช้า, เย็น
- ใช้สีประจำวันในคอลัมน์ซ้าย เพื่อ scan เร็วบนมือถือ
- มี logo เกม/ช่องบริเวณด้านบน
- มี character หรือ mascot รอบกรอบ เพื่อให้ภาพดูสนุก
- เวลาใช้รูปแบบสั้น เช่น `07.00 - 09.30`

## ปัญหาที่ควรปรับ

- รายละเอียดบางแถวแน่นเกินไป โดยเฉพาะชื่อเกมและ set
- ถ้าลดเวลาไลฟ์ ควรทำให้ช่องเวลาเป็น slot ที่ยืดหยุ่น ไม่ fix เฉพาะเช้า/เย็น
- ตัวหนังสือบางจุดเล็กสำหรับมือถือ
- ถ้ามีทั้ง Golden Spatula และ LoL Wild Rift ควรมีสีหรือ icon แยกเกมให้ชัด

## มาตรฐานใหม่ที่แนะนำ

- ทำ data กลางแยก `morning`, `evening`, และ `note`
- ถ้าเป็นตารางรายวัน ใช้ layout เดิมแต่แสดงเฉพาะ slot ของวันนั้น
- ถ้าเป็นตารางรายเดือน ใช้ข้อมูลจาก weekly schedule หลายไฟล์มารวมเป็น calendar view
- คงขนาด export หลักเป็น 1080x1080 สำหรับโพสต์ social และ 1920x1080 สำหรับ live cover
