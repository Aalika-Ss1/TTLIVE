# Starting Soon Scene

ใช้ก่อนขึ้นไลฟ์ เพื่อบอกว่ากำลังจะเริ่ม

## ขนาด

- 1920x1080 สำหรับ OBS/Live Studio
- 1080x1080 สำหรับโพสต์ social ถ้าต้องการ

## องค์ประกอบ

- Logo TTLIVE
- ข้อความหลัก: `กำลังจะเริ่มไลฟ์`
- เกม/หัวข้อ
- เวลาเริ่ม
- ภาพตัวละครหรือ icon เกม
- QR/link/social handle ถ้ามี

## Text Example

```text
กำลังจะเริ่มไลฟ์
Golden Spatula
เริ่ม 20:00 น.
```

## ห้าม

- ใส่ข้อความยาว
- เปลี่ยน layout ทุกครั้ง
- ใช้พื้นหลังที่ทำให้อ่านตัวหนังสือยาก
