# Live Scenes

มาตรฐานภาพและ layout ที่ใช้ก่อนขึ้นไลฟ์ ระหว่างไลฟ์ และหลังลงไลฟ์

## Scene Types

```text
starting-soon      ใช้ก่อนเริ่มไลฟ์ 3-10 นาที
in-live-overlay    ใช้ระหว่างไลฟ์
break-screen       ใช้ตอนพักชั่วคราว
offline-recap      ใช้หลังลงไลฟ์หรือโพสต์ recap
clip-cover         ใช้เป็นปกคลิปสั้น
```

## กฎหลัก

- ข้อความต้องอ่านได้บนมือถือ
- logo ต้องอยู่ตำแหน่งเดิม
- อย่าใส่ข้อมูลเยอะในภาพเดียว
- สีและฟอนต์ต้องอิง `01_brand/`
- ไฟล์ final เก็บใน `08_outputs/`
