# Offline / Recap Scene

ใช้หลังลงไลฟ์หรือทำโพสต์ recap

## องค์ประกอบ

- ขอบคุณที่มาดู
- เกม/หัวข้อที่เล่น
- highlight สั้นๆ
- นัดหมายไลฟ์ครั้งต่อไป
- call to action เช่น ดูคลิปย้อนหลัง / ติดตามไว้

## Text Example

```text
ขอบคุณที่มาดูไลฟ์
วันนี้ Golden Spatula สนุกมาก
เจอกันรอบหน้า 20:00 น.
```

## ใช้ต่อยอด

- ทำโพสต์หลังไลฟ์
- ทำ story
- ทำ cover สำหรับ recap clip
