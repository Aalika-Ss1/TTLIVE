# Clips

พื้นที่สำหรับวางแผนคลิปที่ตัดจากไลฟ์

## Pipeline

```text
live moment -> clip candidate -> edit -> caption -> upload -> performance report
```

## กฎสำหรับช่วงเริ่มต้น

- 1 live ควรมี clip candidate อย่างน้อย 1 ช่วง
- เริ่มจากตัดคลิปสั้น 20-60 วินาที
- อย่าตัดหลายคลิปเกินไปจนกระทบการไลฟ์
- คลิปที่ดีควรมี hook ภายใน 1-3 วินาทีแรก
