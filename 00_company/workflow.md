# Workflow

## Content Pipeline

```text
idea -> planned -> scripted -> designed -> scheduled -> live -> clipped -> posted -> reported
```

## รายละเอียดสถานะ

| สถานะ | ความหมาย |
| :--- | :--- |
| idea | มีไอเดียแต่ยังไม่เลือกทำ |
| planned | เลือกทำแล้วและมีวันเวลา |
| scripted | มีหัวข้อ คำพูด หรือ flow |
| designed | มีภาพปก/ภาพตาราง/asset แล้ว |
| scheduled | ตั้งเวลาโพสต์หรือประกาศแล้ว |
| live | ไลฟ์เสร็จแล้ว |
| clipped | ตัดคลิป highlight แล้ว |
| posted | โพสต์ลง platform แล้ว |
| reported | สรุปผลแล้ว |
