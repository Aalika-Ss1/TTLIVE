# Live Analytics

เก็บรายงานผลของแต่ละไลฟ์ เพื่อใช้ตัดสินใจเรื่องเวลา เกม หัวข้อ และคลิปที่ควรตัด

## หลักการ

- กรอกให้สั้นพอทำได้จริง
- กรอกหลังไลฟ์ภายในวันเดียวกัน
- เน้นสิ่งที่ใช้ปรับรอบถัดไป

## ชื่อไฟล์แนะนำ

```text
YYYY-MM-DD_game_live-report.md
```

ตัวอย่าง:

```text
2026-06-15_wild-rift_live-report.md
```
